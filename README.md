# Video Editor

This is a video editor.

## Installation

```shell
$ pip install -r requirements.txt
$ python3 editor.py
```

## Keyboard Shortcuts

| Shortcut              | Action                        |
|-----------------------|-------------------------------|
| Space                 | Play / Pause                  |
| S                     | Select Mode                   |
| C                     | Cut Mode                      |
| M                     | Move Mode                     |
| D / Delete            | Delete Clip                   |
| X                     | Extract Clip                  |
| ArrowLeft             | Jump 1 Frame Backward         |
| ArrowRight            | Jump 1 Frame Forward          |
| J                     | Jump 1 Second Backward        |
| K                     | Jump 1 Second Forward         |
| H                     | Jump 10 minutes Backward      |
| L                     | Jump 10 minutes Forward       |
| P                     | Jump To Next Empty Part       |
| +                     | Increase Audio Offset         |
| -                     | Decrease Audio Offset         |
| R                     | Hide / Show Clip Bin          |
| T                     | Hide / Show Video Preview     |
| Y                     | Hide / Show Timeline          |
| E                     | Extend Clip From Both Ends    |
| Ctrl+ArrowRight       | Extend Clip From End          |
| Ctrl+ArrowLeft        | Shrink Clip From End          |
| Ctrl+Shift+ArrowRight | Shrink Clip From Start        |
| Ctrl+Shift+ArrowLeft  | Extend Clip From Start        |
| Ctrl+R                | Render Project                |
| Ctrl+O                | Open Project File or Add Clip |
| Ctrl+S                | Save Project                  |
| Ctrl+Shift+S          | Save Project As               |
| Ctrl+Z                | Undo                          |
| Ctrl+Y                | Redo                          |
